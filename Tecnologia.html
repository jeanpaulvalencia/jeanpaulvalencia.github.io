Tecnología – Excel (Funciones y fórmulas)

Resumen del tema

Excel es una hoja de cálculo que permite realizar cálculos y análisis de datos de manera eficiente. Utiliza funciones como =SUMA() para sumar, =PROMEDIO() para calcular la media y =SI() para crear condiciones lógicas. Estas herramientas son esenciales para el manejo de grandes cantidades de información.

Pregunta tipo ICFES

¿Qué fórmula en Excel permite calcular el promedio de los valores entre A1 y A5?

  • A) =SUMA(A1:A5)/5
  • B) =MEDIA(A1:A5)
  • ✅ C) =PROMEDIO(A1:A5)
  • D) =SI(A1:A5)
